What We Used to Build This
HTML: For structured, clean, and semantic web layouts.
CSS: For all the styling, custom layouts, and making sure the design looks good and fits perfectly on both desktop monitors and mobile phone screens.
JavaScript: Used for managing the interactive elements, DOM manipulation, handling dynamic states, and keeping user transitions running smoothly.

---

Features We Are Proud Of
Fully Responsive Layout: We spent a lot of time adjusting the media queries so the layout dynamically adapts without breaking.
Interactive Components: Implemented custom JavaScript logic to drive interactive elements and enhance user engagement.
Clean Folder Structure: Organized the code neatly separating styles, script behaviors, and media assets so it's easy to read and maintain as a team.
